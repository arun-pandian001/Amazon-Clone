# 🛒 Amazon Clone - Vanilla JavaScript

A fully functional Amazon Clone built with HTML, CSS, and Vanilla JavaScript. This project demonstrates E-commerce logic like product filtering, cart management, and local storage integration.

---

## 🚀 Live Demo

amazon-clone-khaki-tau.vercel.app

---

## ✨ Features
* Dynamic Product Listing: Products are rendered dynamically from a JSON-like data structure.
* Add to Cart: Users can add multiple items to the cart.
* Real-time Price Calculation: The cart total and shipping costs are calculated automatically.
* Local Storage: Cart items persist even after refreshing the page.
* Responsive Design: Optimized for both Desktop and Mobile users.
* Order Tracking: Basic UI for tracking orders and delivery dates.

---

## 🛠️ Technologies Used
* Frontend: HTML5, CSS3 (Flexbox & Grid)
* Scripting: Vanilla JavaScript (ES6+)
* Deployment: Vercel / GitHub Pages
